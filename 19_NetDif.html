Network Epidemic Models

Prof. Dr. Juanjo Manjarín

  • 1 Degree-block Approximation
  • 2 Network Epidemics


In this document we are going to deal with an introduction to the epidemic models in networks. Note that the first main idea we need to keep in mind is that the infection can only be produced right in the neighborhood of an infected node

It is also true, however, that due to the different origins of any disease, some oubreaks may occur in different places of the network at the same or almost the same time. Then the dynamics of the network can be rather complicated with random nodes getting infected and then spreading the disease, others getting recovered, others getting retired from the network, thus breaking connections, and so on.

The first main impact in the equations of the classical epidemic models is that if an individual has \(\langle k \rangle\) contacts with a probability of transmission of \(\beta\), the average number of infections in a time interval \(dt\) is

\[\begin{equation} \langle k \rangle \beta\frac{S(t)I(t)}{N} \end{equation}\]

which means that there is a factor of \(\langle k \rangle\) that will appear in all our formulas and that we have not previously considered. For example,

\[\begin{equation} R_0 = \frac{\beta \langle k\rangle}{\gamma} \end{equation}\]

However, even this is not enough, since we already know that the topology of the network makes a difference when it comes to specify the dynamics in and of the network, for example, in a power-law network, the average degree is important, but the degree exponent is even more since it determines the regime of the network and if whether or not the average degree exists.

1 Degree-block Approximation

In this formalism we consider that nodes with the same degree are equivalent, this allow us to write the SI rate of variation for the infected as

\[\begin{equation} \frac{d i_k}{dt} = \beta (1-i_k)k\Theta_k \end{equation}\]

where we have replaced the average degree by the degree of node and we have to consider the density \(\Theta_k\) which represents the fraction of infected neighbors of a susceptible node \(k\). From this equation we can obtain

\[\begin{equation} i(t) = i_0\left(1 + \frac{\langle k\rangle^2 - \langle k\rangle}{\langle k^2\rangle - \langle k\rangle}(e^{t/\tau} - 1)\right) \end{equation}\]

where

\[\begin{equation} \tau = \frac{\langle k \rangle}{\beta(\langle k^2\rangle - \langle k\rangle)} \end{equation}\]

is the characteristic time for the spread of the pathogen. We should make a difference with respect to the type of network we have, then

  • For random networks, where \(\langle k^2\rangle = \langle k\rangle(\langle k\rangle+1)\), this becomes

\[\begin{equation} \tau = \frac{1}{\beta \langle k\rangle} \end{equation}\]

exactly the same as in the SI-model.

  • Scale-free network with \(\gamma\geq 3\), i.e. equivalent to a random network, the elements in the computation of the time are finite and we can compute it as in the SI-model
  • Scale-free network with \(\gamma < 3\), in this case \(\langle k^2\rangle\to\infty\) and then the characteristic time tends to 0, which means that the spread of the pathogen is instantaneous

In the SIS and SIR models, we can find similar expressions for the characteristic times, in fact

\[\begin{equation} \tau^{\rm SIS} = \frac{\langle k\rangle}{\beta\langle k^2\rangle -\gamma\langle k\rangle} \end{equation}\]

and

\[\begin{equation} \tau^{\rm SIR} = \frac{\langle k\rangle}{\beta\langle k^2\rangle -(\gamma+\beta)\langle k\rangle} \end{equation}\]

In these two models we know, however, that the disease will not spread unless the rate between infection to recover is larger than a value, let’s call it \(\lambda\). This value is known as epidemic threshold. We can write this threshold by imposing that the characteristic time must be larger than 0 for the disease to spread, then if

\[\begin{equation} \lambda \frac{\beta}{\gamma} \end{equation}\]

For the SIS-model we obtain

Model Random Network Scale-Free Network
Threshold \(\displaystyle \frac{1}{\langle k\rangle +1}\) \(\displaystyle \frac{\langle k\rangle}{\langle k^2\rangle}\)

in such a way that if the spreading rate is smaller than the threshold, the pathogen just dies, while if it is larger, the epidemic grows until it reaches the endemic state where a fraction \(i(\lambda)\) of the population is always infected.

For the SIR-model we obtain

Model Random Network Scale-Free Network
Threshold \(\displaystyle \frac{1}{\langle k\rangle}\) \(\displaystyle \frac{\langle k\rangle}{\langle k^2\rangle - \langle k\rangle}\)

with some care in both cases with respect to the regime of the power law, i.e. with the value of the degree exponent

2 Network Epidemics

Following Bjonrnstad’s book Epidemics, we can define the following functions

SIR_net <- function(x, beta, gamma, seed = 101){
  # This function generates the values for a SIR model
  N <- dim(x)[1]
  I <- matrix(rep(0,N), nrow = N, ncol = 1)
  S <- matrix(rep(1,N), nrow = N, ncol = 1)
  R <- matrix(rep(0,N), nrow = N, ncol = 1)
  
  set.seed(seed)
  init_inf <- sample(1:N, size = 1)
  I[init_inf, 1] <- 1
  S[init_inf, 1] <- 1
  t <- 1
  while( sum(I[,t-1]) > 0 | t == 1){
    t <- t+1
    neig_inf <- x %*% I[, t-1]
    prob_inf <- 1 - (1-beta)^neig_inf
    
    new_inf <- rbinom(N, S[,t-1], prob_inf)
    new_rec <- rbinom(N, I[,t-1], gamma)
    
    next_s <- S[, t-1] - new_inf
    next_i <- I[, t-1] + new_inf - new_rec
    next_r <- R[, t-1] + new_rec
    
    S <- cbind(S, next_s)
    I <- cbind(I, next_i)
    R <- cbind(R, next_r)
  }
  out <- list(S = S,I = I,R = R)
  return(out)
}

model.netSIR <- function(x){
  # Just turn the output into a dataframe
  S <- apply(x$S, 2, sum)
  I <- apply(x$I, 2, sum)
  R <- apply(x$R, 2, sum)
  time <- 0:(length(S)-1)
  
  output = data.frame(time = time, S=S/max(S), I=I/max(S), R=R/max(S))
  return(output)
}

net_R0 <- function(x, beta, gamma){
  # Computation of R0 for a network
  deg_dist <- apply(x, 2, sum)
  deg_mean <- mean(deg_dist)
  deg_var <- mean(deg_dist^2)
  
  R0 <- beta/(beta+gamma) * (deg_var - deg_mean)/deg_mean
  return(R0)
}

To work with a disease outbreak in a network. The spreading of the disease is modelized using a chain binomial model, which assumes that the epidemics is formed from a sequence of discrete Bernoulli events. In this model, the probability of infection along a S-I edge, is \(1 - (1-\beta)^y\), where \(y\) represents the fraction of infected nodes in the neighbor of the susceptible node.

Also, in this case we can find the value of \(R_0\) as

\[\begin{equation} R_0 = \frac{\beta}{\beta + \gamma}\frac{\langle k^2\rangle - \langle k\rangle}{\langle k\rangle} \end{equation}\]

Let’s create three different models and obtain their corresponding adjacency matrices

# Networks -------------------------------------------------------
set.seed(101)
# ER-model
er_epi <- erdos.renyi.game(50, 0.05, "gnp")
er_am_epi <- as.matrix(get.adjacency(er_epi))

# WS-model
ws_epi <- watts.strogatz.game(1, 50, 3, 0.05)
ws_am_epi <- as.matrix(get.adjacency(ws_epi))

# BA-model
ba_epi <- barabasi.game(50, 0.9, out.dist = c(0.35, 0.45, 0.20), directed = FALSE)
ba_am_epi <- as.matrix(get.adjacency(ba_epi))

Let’s simulate the eopidemics in the three network models, by assuming a probability of infection of 0.3 and a probability of recovering of 0.1

# Probabilities
beta <- 0.5
gamma <- 0.1

# Models
er_sir <- SIR_net(er_am_epi, beta, gamma, 123)
ws_sir <- SIR_net(ws_am_epi, beta, gamma)
ba_sir <- SIR_net(ba_am_epi, beta, gamma)

# Fits
er_fit <- model.netSIR(er_sir)
ws_fit <- model.netSIR(ws_sir)
ba_fit <- model.netSIR(ba_sir)

# Ro
er_R0 <- net_R0(er_am_epi, beta, gamma)
ws_R0 <- net_R0(ws_am_epi, beta, gamma)
ba_R0 <- net_R0(ba_am_epi, beta, gamma)

Once we have these simulations, we can find the final epidemic size and with it (not done here) the fraction of the susceptible population that will scape the infection.
